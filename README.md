# 📺 YouTube Trending Video Analysis

## 📌 Project Description

This project performs **Exploratory Data Analysis (EDA)** on YouTube Trending Videos data to understand the factors that influence video popularity and trending performance.
The analysis explores video categories, views, likes, comments, publishing trends, and audience engagement using Python-based data visualization techniques.

---

## 🎯 Objectives

* Clean and preprocess YouTube trending dataset
* Analyze trending video categories
* Identify most viewed and most liked videos
* Study channel performance and engagement metrics
* Visualize trends using charts and graphs

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset Information

The dataset contains:

* Video Title
* Channel Title
* Category
* Views
* Likes
* Comments
* Publish Time
* Trending Date

---

## 📊 Analysis Performed

* Data Cleaning & Preprocessing
* Category-wise Trending Analysis
* Views, Likes & Comments Analysis
* Top Trending Channels Identification
* Engagement Analysis
* Data Visualization using Bar Charts & Heatmaps

---

## 📈 Key Insights

* Entertainment and Music categories frequently dominate trending videos.
* Videos with higher engagement tend to trend faster.
* Popular channels consistently appear in trending lists.
* Viewer interaction plays an important role in video success.

## 🚀 How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/youtube-trending-analysis.git
   ```

2. Navigate to the project folder:

   ```bash
   cd youtube-trending-analysis
   ```

3. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

4. Open Jupyter Notebook:

   ```bash
   jupyter notebook
   ```

5. Run **youtube1.ipynb**
